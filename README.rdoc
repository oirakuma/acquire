# API list

GET    /games/:id/reset
--------------------------
指定したゲームの状態をリセットする。

POST   /games/:id/put_tile
--------------------------
  タイルを置く。戻り値のresultに結果が含まれている。
  merged、chained、null。
  mergedのときは株券の売却・交換の完了通知が必要。
  chainedのときはホテルチェーンの色の通知が必要。

GET    /games/:id
-----------------
  ゲームの状態を返す。マージが発生した時はmergedに色が入っている。
  自分が完了通知をしたかどうかはuser.mergedにある。

POST   /games/:id/build_chain
-----------------------------
  ホテルチェーンの色を通知し、チェーンを形成する。

POST   /games/:id/purchase_stock
-----------------------------
  株券を購入する。

POST   /games/:id/purchase_done
-----------------------------
  株券の購入を完了する。このAPIを実行すると次のユーザの手番になるように状態が移行する。

POST   /games/:id/sell
-----------------------------
  株券を売却する。

POST   /games/:id/trade
-----------------------------
  株券を交換する。

POST   /games/:id/merge_done
-----------------------------
  マージ時後の株券の処理の完了通知。

POST   /games/entry_name
-----------------------------
  名前を登録する。

GET    /games
-----------------------------
  ゲーム一覧を取得する。
